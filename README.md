# 🐾 Flask Pet API

## 📌 Description
A simple Flask web app that fetches and displays random pet images like cats, dogs, foxes, and pandas using public APIs.

## 🚀 Features
- Get random cat images 🐱
- Get random dog images 🐶
- Get random fox images 🦊
- Get random panda images 🐼
- Simple UI with loader animation

## 🛠️ Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- External APIs

## 📂 Project Structure
flask-pet-api/
│── app.py  
│── requirements.txt  
│── templates/
│── static/

## ▶️ How to Run Locally

1. Clone repo:
git clone https://github.com/nashrasadaff/flask-pet-api.git

2. Navigate:
cd flask-pet-api

3. Install dependencies:
pip install -r requirements.txt

4. Run the app:
python app.py

5. Open in browser:
http://127.0.0.1:5000
